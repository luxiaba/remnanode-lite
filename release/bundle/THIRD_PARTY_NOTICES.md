## Xray-core (`rw-core`)

`lib/rw-core` is the unmodified `xray` executable from XTLS/Xray-core
`v26.6.27`, commit `45cf2898ab12e97a55dd8f1f3d78d903340bdc9e`. It is renamed
so the service can distinguish the managed runtime from a system-wide Xray
installation. Xray-core is licensed under MPL-2.0. The corresponding source is:

https://github.com/XTLS/Xray-core/tree/45cf2898ab12e97a55dd8f1f3d78d903340bdc9e

ipverse/as-ip-blocks commit `56d021c7536afb15317155e45b57e7b5c87a4700`.
The source dataset is dedicated to the public domain under CC0-1.0:

https://github.com/ipverse/as-ip-blocks/tree/56d021c7536afb15317155e45b57e7b5c87a4700

The release manifest records the exact source archive and generated output
digests. No network source is consulted when the installed service starts.
